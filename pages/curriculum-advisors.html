Curriculum Advisors are part of a team that provides the oversight, vision, and leadership for a particular set of lessons. Committees can be reached by email using the addresses listed below, or in GitHub by using the @ tag to reference the appropriate Curriculum Advisory team (e.g. "@datacarpentry/curriculum-advisors-astronomy") in an Issue, Pull Request, or comment. More information about the Curriculum Advisory Committees can be found in The Carpentries Handbook.
